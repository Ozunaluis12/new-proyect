AHORA (5 minutos)

GOOGLE MAPS API (10 minutos)
  1. https://console.cloud.google.com/
  2. Crear proyecto
  3. Habilitar "Directions API" + "Geocoding API"
  4. Crear API Key
  5. Copiar en: lib/constants/app_constants.dart

FIREBASE (15 minutos)
  1. https://console.firebase.google.com/
  2. Crear proyecto "loginova"
  3. Agregar app Flutter
  4. Descargar google-services.json → android/app/
  5. Descargar GoogleService-Info.plist → ios/Runner/
  6. Actualizar firebaseProjectId en app_constants.dart

TESTING (10 minutos)
  1. flutter run
  2. Probar ubicación
  3. Probar rutas
  4. Probar notificaciones
